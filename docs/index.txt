Basic installation
******************

First, you will need to do the :doc:`standard snstallation </enduser/installation/index>`

If you encounter some errors, please check :doc:`the errors and solutions page </enduser/errors/index>`

Then, you will need to:

* Configure Domogik over the administration interface
* Install a control user interface like Domoweb or Domodroid

Uninstallation
**************

If you want to uninstall Domogik, you will need to :doc:`use the uninstall tool </enduser/installation/uninstall>` .

Advanced installation tutorials
*******************************

The following tutorials are not mandatory to use Domogik. You can pick just what you need for what you want.

* :doc:`Install Domogik from a script </enduser/installation/install_from_a_script>`
* :doc:`Use postgresql instead of mysql </enduser/installation/postgresql>`
* Configure external access to Domogik administration interface
* :doc:`Install Domogik in a python virtualenv </enduser/installation/virtualenv>`
* :doc:`Configure SSL </enduser/installation/ssl>`

* :doc:`Overview </technical/rest/index>`

* :doc:`/map </technical/rest/map>`: A dynamicly generated list of all possible REST urls
* :doc:`/datatype </technical/rest/datatype>`: The output of all known datatypes by domogik

        * :doc:`/device/params </technical/rest/device_params>`: list the parameters associated to a device type

        * :doc:`/sensor </technical/rest/sensor>`: manage sensors (list, update)
        * :doc:`/sensorhistory </technical/rest/sensorhistory>`: list sensorhistory values
    * Command sending
        * :doc:`/cmd </technical/rest/cmd>`: execute an already created command
* Accounts and persons
    * :doc:`/account </technical/rest/account>`: manage accounts (list, create, update, delete, authenticate)
    * :doc:`/person </technical/rest/person>`: manage persons (list, create, update, delete)
* Old Domogik 0.2/0.3 devices migration
    * :doc:`/device/old </technical/rest/device_old>`: list the old devices (created with Domogik <= 0.3)
